Design decisions
ThemesLight (#f5f0e8 bg / white cards) + Dark (#0d0a08) — auto-switches with prefers-color-scheme
Sport & league navRemoved from sticky header. "Filter ⊞" button opens a bottom sheet: sport icon grid on top, league checkbox list below
Match cards2-column grid. Live → Results → Upcoming sections. Cards show: league, both team names stacked, score, status badge
Spoiler toggleKept as full-width card between header and grid. Amber when scores showing, blue when hidden. Scores blur rather than disappear
Date navMoved into header row (compact ‹ Mar 15 ›) — saves vertical space
What stays the sameAll data fetching, API logic, standings/team pages, tennis/F1 views — no backend changes
